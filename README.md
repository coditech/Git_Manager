# Git_Manager
Simple Node Program to clone all students repositories. (Could be done in .SH)


## Setup 
```sh
    git clone https://github.com/coditech/Git_Manager.git
    cd Git_Manager
    cd 00_GitManager
    npm install
```

## Use
```js
    node ./GitManager.js
```

It should clone all submissions folder.
